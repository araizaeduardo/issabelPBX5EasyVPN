# Changelog

## [1.0.0] - 2025-12-06
- Implementación inicial del módulo EasyVPN para Issabel 5.
- Scripts CLI para crear, revocar, regenerar perfiles y consultar estado OpenVPN.
- Instalador `install.sh` con opción `--with-server` para desplegar OpenVPN/EasyRSA.
- Interfaz web con auto-refresh de clientes conectados y control del servicio.
